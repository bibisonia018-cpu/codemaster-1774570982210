# 🛡️ Secret Chat App (End-to-End Encrypted)

تطبيق دردشة سرية آمن 100%. يستخدم التشفير من طرف إلى طرف (E2EE) باستخدام تشفير `AES`. الخادم لا يرى الرسائل، وقاعدة البيانات تقوم بحذفها تلقائياً بعد ساعة.

## 🚀 كيفية التشغيل (Setup Instructions)

### 1. تشغيل الخادم (Backend)
1. افتح الطرفية (Terminal) وادخل لمجلد `backend`: `cd backend`
2. قم بتثبيت الحزم: `npm install`
3. تأكد من تشغيل خادم MongoDB لديك محلياً على البورت 27017، أو أضف ملف `.env` وضع فيه `MONGO_URI`.
4. تشغيل الخادم: `npm start`
الخادم سيعمل على المنفذ `5000`.

### 2. تشغيل واجهة المستخدم (Frontend)
1. افتح طرفية جديدة وادخل لمجلد `frontend`: `cd frontend`
2. قم بتثبيت الحزم: `npm install`
3. قم بتشغيل تطبيق React: `npm start`
التطبيق سيعمل على متصفحك في `http://localhost:3000`.

## 🔐 كيف يعمل التشفير (How Security Works)
- يقوم المستخدمون بإدخال **كلمة سر التشفير (E2E Encryption Password)**.
- يتم استخدام `crypto-js` لتشفير الرسالة في المتصفح قبل إرسالها.
- تذهب الرسالة المشفرة (Ciphertext) إلى Node.js وتُحفظ في MongoDB.
- المستلم يستقبل الرسالة المشفرة، ويستخدم كلمة السر التي لديه لفك تشفيرها في متصفحه.
- إذا قام شخص بالدخول بكلمة سر تشفير خاطئة، سيظهر له نص غير مقروء.
- خاصية `expires` في `Message.js` تقوم بإخبار MongoDB بحذف الرسالة من السيرفر نهائياً بعد مرور ساعة.
